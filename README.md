# House Price Prediction

A machine learning project that predicts house prices using the Ames Housing Dataset.

## Models Evaluated

* Linear Regression
* Random Forest Regressor
* XGBoost Regressor

## Results

| Model                          | RMSE   |
| ------------------------------ | ------ |
| Linear Regression (1 Feature)  | 58,472 |
| Linear Regression (5 Features) | 39,763 |
| Random Forest                  | 28,883 |
| XGBoost                        | 25,829 |

Final Model:

* Dollar RMSE: 26994.5221701851
* Log RMSE: 0.13238959751083887

## Techniques Used

* Exploratory Data Analysis (EDA)
* Missing Value Imputation
* One-Hot Encoding
* Feature Engineering
* Cross Validation
* Feature Importance Analysis
* XGBoost Regression

## Dataset

Ames Housing Dataset
